C. 컬러 (Color System)

브랜드 고유 정체성을 확립하고 일관된 정보 계층을 전달하기 위한 핵심 색상 규칙 및 인터랙션 상태별 추출 가이드를 정의합니다.

3-1 브랜드 컬러 가이드라인 (Brand Color System)

브랜드 컬러 정의 및 활용 방식

브랜드 컬러란 브랜드를 나타내는 고유색으로 브랜드 전체의 통일감을 부여하며, 사용자에게 끊임없이 같은 메시지를 전달하기에 효과적입니다.
인터페이스 설계 시 주로 Primary Color 1가지와 Secondary Color 1가지 조합 구성을 제안합니다. 다수의 컬러를 난무하여 사용하는 것보다 소수의 컬러를 엄격하게 제한하는 것이 브랜드성을 명확히 확립하면서 콘텐츠 중심의 깔끔한 디자인 전개를 쉽게 도와줍니다. 화면이 너무 단조롭게 느껴진다면 정의된 브랜드 컬러에 채도(Saturation)와 명도(Brightness) 단계를 확장 부여하여 풍부한 컴포넌트 표현을 보완할 수 있습니다.

시스템 제안 코어 핵심 브랜드 팔레트 예시
Primary Color

주 색상 (메인 정체성 타겟팅)

브랜드 정체성의 중심을 잡으며 UI 전반의 주요 액션 단추, 핵심 하이라이트에 집중 배치합니다.

Secondary Color

보조 색상 (서브 포인트 레이아웃)

주 색상을 시각적으로 보완하며 프로모션 배지, 특수한 기능적 분기점 요소에 한정 사용합니다.

#ff6e40

브랜드 컬러 배합 및 점유 비율 규칙

화면 구조 설계 시 Primary Color가 Secondary Color보다 시각적 점유 비율이 확연히 높아야 합니다. 이 규칙이 어긋나 보조 색상이 과도하게 남발될 경우, 사용자는 인터페이스 요소 간의 중요도 및 주요 유도 동선(CTA)에 대한 인지적 혼란을 겪을 수 있습니다.
또한, 주색과 보조색을 선정할 때 색상 간의 조화도 매우 중요합니다. 컬러 휠(색상환) 도구를 사용하면 수학적으로 계산된 보색, 유사색 배열을 매핑할 수 있어 컬러 충돌에 인한 시간 지체를 획기적으로 줄여줍니다. 본 가이드라인의 #3d5afe와 #ff6e40 관계는 보색(Complementary) 관계로 대비 효과를 극대화해 웹상에서 주목도를 유도하기 좋은 궁합의 예시입니다.

시각적 면적 점유 비율 시뮬레이터 (60:10 비율 스펙 권장)
Primary (85%)
Sec (15%)
Primary Color (#3d5afe): 고대비 핵심 액션 인쇄 영역 지배적 점유
Secondary Color (#ff6e40): 시선 고정을 위한 포인트 강조 및 보색 보완

인터랙션 컬러 (Interaction Color Scale) 구축 방법

인터랙션 컬러는 마우스 오버(Hover), 클릭(Active) 등 컴포넌트의 유동적 상태 변화를 명확하게 피드백하기 위해 정의합니다. 디폴트 기본 컬러에서 1~2톤 정도 밝거나(Lighten) 어두운(Darken) 스펙트럼 수치를 제안합니다.
고정된 1~2개 고정색 대비 기본 디폴트 값을 기준으로 상하 명도 단계를 4~5단계 확장해 놓으면 훨씬 유연하고 세련된 시각 디자이너 처리가 가능해집니다. Lighten 축(왼쪽)으로 갈수록 점점 백색에 가까워지며 숫자가 커지고, Darken 축(오른쪽)으로 진입할수록 어두워지며 단계 수치가 커집니다.

마우스 호버(Hover) 상태별 실제 마크업 피드백 예시

마우스 호버 시 인터랙션 컬러 변화를 의도대로 적용하면 사용자가 현재 링크나 액션 영역 위에 마우스를 정확하게 위치시키고 있는지 직관적으로 알려줄 수 있어 가독성과 사용성을 크게 늘립니다.
아래 예시는 호버 발생 전 기본 스펙으로 Default 컬러(#3d5afe)를 고정 매핑하고, 마우스 포인터가 진입했을 때 Interaction Color Lighten-1 단계(#667bff)로 반응형 스위칭 체계를 적용한 실무 버튼 설계 모형입니다. Right 버튼에 마우스를 직접 올려 변화를 검증할 수 있습니다.

라이브 버튼 호버 상태 시뮬레이터
Normal State (기본 대기) #3d5afe (Default)
Hover State (마우스 오버 작동) #667bff (Lighten-1)
웹 접근성(WCAG 2.1) 컬러 매칭 팁
Primary 컬러인 #3d5afe와 확장형 인터랙션 스케일 가이드를 활용할 시, 백색 배경 대비 본문 텍스트 규격은 가독성을 위해 최소 AA 스펙(명도비 4.5:1 이상)을 준수해야 합니다. 버튼 내부 흰색 텍스트와 주색 배합은 강력한 고대비를 이루어 최적의 가독성을 제공합니다.
